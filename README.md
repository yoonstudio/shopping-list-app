# 쇼핑 관리 앱

매일 10-20개 정도의 쇼핑을 효율적으로 관리할 수 있는 사이드용 웹 애플리케이션입니다.

## 주요 기능

### 기본 기능
- **쇼핑 추가**: 입력 필드에 쇼핑을 입력하고 카테고리를 선택하여 추가
- **쇼핑 완료**: 체크박스를 클릭하여 완료/미완료 상태 토글
- **쇼핑 수정**: 연필 아이콘을 클릭하여 내용과 카테고리 수정
- **쇼핑 삭제**: x 버튼을 클릭하여 쇼핑 삭제

### 진행률 관리
- 전체 쇼핑 대비 완료된 쇼핑의 진행률 표시
- 진행률 바와 퍼센트로 시각적 표현
- 완료/전체 개수 텍스트 표시

### 카테고리 관리
- 3가지 카테고리: 메인, 사이드, 간식
- 카테고리별 색상 구분
  - 메인: 파란색
  - 사이드: 초록색
  - 간식: 주황색
- 카테고리별 필터링 기능

### 데이터 저장
- Supabase를 이용한 클라우드 데이터 저장
- 다크 모드 설정은 localStorage 사용
- 어디서든 데이터 동기화 가능

## 사용 방법

1. **쇼핑 추가하기**
   - 입력 필드에 쇼핑을 입력
   - 드롭다운에서 카테고리 선택
   - "추가" 버튼 클릭 또는 Enter 키 입력

2. **쇼핑 완료하기**
   - 쇼핑 항목의 체크박스 클릭
   - 완료된 항목은 취소선과 회색으로 표시

3. **쇼핑 수정하기**
   - 연필 아이콘 클릭
   - 내용과 카테고리 수정
   - "저장" 버튼 클릭 또는 Enter 키 입력
   - "취소" 버튼 또는 Escape 키로 취소

4. **쇼핑 삭제하기**
   - x 버튼 클릭
   - 확인 대화상자에서 "확인" 클릭

5. **카테고리 필터링**
   - 전체, 메인, 사이드, 간식 버튼을 클릭하여 필터링

## 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 모던 스타일링, 애니메이션, 반응형 디자인
- **Vanilla JavaScript**: 순수 자바스크립트
- **Supabase**: 클라우드 데이터베이스

## 실행 방법

1. `index.html` 파일을 브라우저에서 열기
2. 또는 로컬 서버를 실행하여 접속

```bash

python -m http.server 8000

# 브라우저에서 http://localhost:8000 접속
```
