# 할 일 관리 앱

매일 10-20개 정도의 할 일을 효율적으로 관리할 수 있는 개인용 웹 애플리케이션입니다.

## 주요 기능

- 할 일 추가/수정/삭제
- 완료 체크 및 진행률 표시
- 카테고리 분류 (업무/개인/공부)
- 카테고리별 필터링
- 다크 모드 지원
- localStorage 데이터 자동 저장
- 반응형 디자인

## 기술 스택

- HTML5 / CSS3 / Vanilla JavaScript
- localStorage API

## 실행 방법

```bash
python -m http.server 8000
# http://localhost:8000 접속
```
